Anti-Corona Jet Fighter

  • Originality: It's easy to play and fun.
  • Technical Merit: Nothing special.
  • Postmortem: I leaned how to to deal with timer and group collision.
    The hardest part is to modify the object size making sure an object collides with others reasonably.
    One big problem is set up 2 state scene at the end, when everything completed is not a good idea.
    The easiest part is all game physics are handled by Phaser framework and it saved a lot of development time.
  • Assets:
    • Viruses, vaccines and airplane: Freepik (Heavily edited by myself.)
    • Background: Freepik
    • Sound Effects: Free Sound ORG
  • Prototype Closest Other Game: - Inspired by my previous game and Corona Outbreak.
    - Very limited code derived from my previous game.
  • High Concept: Control the plane, avoid all viruses and try to get many vaccines as you can to save lives.
  • Theme: Doom day of the world.
  • Mandated Variety: N/A
  • Prototype Goal: This is a final release. (Patches will be released if bugs found).
  • Player Experience Goals: I hope people to see it fun and can relax with it.
  • Gameplay: User arrow buttons to move the plane.
    Super Virus inverts movement for 3 seconds.
    Vaccine saves 1 live.
    Pilot Vaccine save 1 pilot.
    Super Vaccine cured super virus and makes you immortal in 5 seconds.
  • Strategies: Try to take super-vaccine as soon as possible. It will make you immortal for 5 seconds and recover the pilots.
  • Story/Setting/Premise: People all over the world were infected except the last 500 people on this plane. You have to bring them to a safe place.
  • Target Audience: This game targets people from 15-30.
  • Play Time: It depends on players. Should be 1 minute - 15 minutes. If they good, it would take about 45 minutes to save all people.
